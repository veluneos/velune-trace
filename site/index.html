로컬에서 바로 실행하세요.

VeluneOS의 로컬 증거 엔진 Velune Trace v0.4.0을 내려받아 MCAP 로그에서 토픽 프로파일, 증거 구간 및 재현 가능한 Core Report Bundle을 생성할 수 있습니다.

Version v0.4.0 Package 174,259 bytes Validated Ubuntu 24.04 · Python 3.12
STEP 01

패키지 다운로드

테스트, 웹사이트 소스, Git 메타데이터 및 원본 MCAP을 제외한 명시적 공개 파일 69개로 구성된 로컬 실행 패키지입니다.

ZIP 다운로드
STEP 02

무결성 확인 후 설치

ZIP과 SHA256SUMS를 같은 디렉터리에 내려받고 압축을 풀기 전에 패키지 해시를 검증하세요. 

sha256sum -c SHA256SUMS
전체 설치 안내
릴리스 노트 및 자산 SHA256SUMS Release Manifest GitHub 저장소

Velune Trace는 로컬에서 실행되며 텔레메트리나 원본 로그의 자동 업로드를 수행하지 않습니다. 관찰된 증거와 차이를 구조화하지만 근본 원인, 과실, 책임, 안전성 또는 회귀 여부를 자동으로 판정하지 않습니다.

SHA-256: d719e7d03be8a6963c3e2141b846064c56805d9b2c288dc6683bfa86ebc83ad6

전체 로그를 직접 뒤지지 마세요.

대규모 로봇 로그에서는 엔지니어가 토픽, 타임스탬프, 데이터 공백, 지터, 명령과 움직임을 직접 추적해야 합니다. VeluneOS는 조사 범위를 좁히고 사람이 검토할 수 있는 재현 가능한 증거 리포트를 생성합니다.

로컬 우선

원본 로그를 중앙 서버에 업로드하지 않고 MCAP과 ROS2 기반 증거를 처리합니다.
